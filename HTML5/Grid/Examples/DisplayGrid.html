Grid

Com o display: grid; é possivel definirmos colunas e linhas para organizarmos os elementos que estiverem dentro do grid.

  • grid-tamplate-columns
    • Define o total de colunas e o tamanho de cada uma
  • fr
    • fr é uma uniade fracionária que tera como objetivo distribuir o espaço restatne, entre os elementos do grid.
  • gap
    • Cria uma distância entre os elementos do grid, tanto horizontal quanto vertical.
