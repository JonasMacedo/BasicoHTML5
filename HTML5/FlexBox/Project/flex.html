Flex

  • Flex-grow: 1
    • Define se o elemento deve crescer para ocupar todo o espaço vazio.
    O valor 0(zero) impede o crescimento do elemento.
    O valor 1(um) define o crescimento, em fracionario FR do CSS.
  • Flex
    • propriedade dos elementos flexiveis.
  • Flex-basis: 200px;
    • Define o valor inicial antes de distribuir o espaço.
  • Flex-shrink: 0;
    • Caso exista um valor de base, o flex-shrink ira definir se o elemento interno base pode ser reduzido ou nao.
    O valor 0(zero) define que ele nao pode ser reduzido.
