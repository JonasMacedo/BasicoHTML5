Fluxo do Layout

O fluxo do layout no HTML ocorre conforme o modo de escrita definido.
Por padrão, de cima para baixo da esquerda para a direita.

  • Span
    • Tag generica que nao possui nenhum estilo pré-difinido/semantica. É equivalente a uma div, mas sem o display block


    Diplay inline e block

    Define como a caixa(BoxModel) irá se comportar.

  • Inline
    • Respeita o fluxo da escrita sem iniciar uma nova linha, não é possivel definir valores de width, height, margin e etc.
    É o estilo padrao.

  • Block
    • Inicia uma nova linha e nao permite que outros elementos sema posicionados em sua linha.
    Aceita todas as propriedades do box model.
    Estilo inicial de elemeots como h1, p, div e outros.


    None e Inline-block

  • None
    • Remove o elemento completamente da tela.

  • Inline-block
    • O elemento continua inline, mas passa a receber as propriedades do box model.
